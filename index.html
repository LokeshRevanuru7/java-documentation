Object Oriented Programming

Java is known as an object oriented programming language.

This means that it is easy to represen entities as objects by using classes and encapsulation.

An example of this might be a student classes torepresent a student 

                
                    public class Student{
                        
                        private String name;
                        private int age;

                        
                        public Student(String name, int age){
                            this.name;
                            this.age;
                        }

                        
                        public String getName(){
                            return name;
                        }

                        
                        public void setName(string name){
                            this.name=name;
                        }
                    }
